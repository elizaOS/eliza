You want to build an AI agent. Maybe it handles customer queries, trades on-chain, automates your workflow or does something nobody's thought of yet.

The problem isn't the idea, it's the execution. Building an agent means wrestling with AI model APIs, managing conversations, handling memory and state, wiring up plugins, and figuring out how to make it all work together. Then, once you've built something that runs on your laptop, you need to get it live, which means operating servers, databases, networking and managing billing.

It's too much. Great ideas get abandoned not because they weren't good enough, but because building and deploying agents is still too hard.

Eliza Labs learnt this through establishing the largest open source agent developer community with thousands of people getting hands on with elizaOS. We built Eliza Cloud to solve these problems, and open the doors for non-technical creators to build the agentic web.

## Build agents, not infrastructure

Eliza Cloud is a low-code platform for building and deploying AI agents. You focus on what your agent does. We handle everything else.

- **Visual agent builder**: Define your agent's personality, capabilities, and behavior through an intuitive interface. No need to manually configure every detail; common patterns work out of the box.
- **Pre-built integrations**: Connect to Discord, Telegram, Twitter, web chat, and more with a few clicks. Your agent works across platforms without custom integration code.
- **Unified AI access**: One account gives you access to text, image, video, and voice AI capabilities. No juggling provider accounts. No managing multiple API keys. Credits work across everything.
- **Built-in memory and state**: Conversations, context, and knowledge are handled automatically. Your agent remembers. It learns. It maintains state, all without you configuring databases or vector stores.
- **Plugin ecosystem**: Extend your agent with on-chain capabilities, knowledge bases, tools, and more. The open-source elizaOS plugin ecosystem works seamlessly.

You describe what you want your agent to do. We handle the complexity of making it actually work.

## From idea to live agent, fast

Whether you're a developer or a creator with no technical background, Eliza Cloud gets your agent live quickly.

**For developers:** Deploy with a single command.

```bash
npx eliza-cloud deploy
```

**For non-technical creators:** Use our visual interface to build and launch your agent without writing code.

The real magic happens before deployment. Eliza Cloud gives you a complete environment to bring your agent to life. Build your agent's personality and capabilities through simple forms and visual tools. Test it, refine it, and see it working all before you go live.

When you're ready to launch, we handle everything behind the scenes. Servers, databases, security, scaling. Your agent gets a URL. It's serving users. No technical expertise required.

If you want to add custom logic or advanced features, you can write code. But you don't have to. We've built the common patterns so you can focus on what makes your agent unique, whether you're coding it or configuring it.

## Open source underneath

Here's what matters most: Eliza Cloud runs the full open-source elizaOS runtime.

Your agents aren't trapped. The code you write isn't in some proprietary format that only works on our platform. If you ever want to self-host, take your agent to another provider, or run it on a Raspberry Pi in your garage then you can.

This means when you own your code and can move it anywhere, you build with confidence. No vendor lock-in means you'll invest more deeply in making your agent great. But moreover, as agents shift from responsive to autonomous, you have the open foundations needed for secure agentic commerce.
