# Cloud Docs Notes

This directory holds engineering notes and WHY-focused implementation docs for
the Cloud package. Public user-facing documentation lives in
`cloud/packages/content`.

Keep files here linked from `cloud/README.md` only when they exist in this
directory. For API shapes, model catalogs, auth headers, rate limits, and
pricing, prefer the public docs or the generated API Explorer as the canonical
source.
