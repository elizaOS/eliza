# @elizaos/plugin-code

Coder tools for ElizaOS: filesystem operations, shell execution, and git commands.

This plugin ships **TypeScript**, **Python**, and **Rust** implementations.

## Configuration

- `CODER_ENABLED` (default: `false`)
- `CODER_ALLOWED_DIRECTORY` (required)
- `CODER_TIMEOUT` (default: `30000`)
- `CODER_FORBIDDEN_COMMANDS` (optional, comma-separated)
