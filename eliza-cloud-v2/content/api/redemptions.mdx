# Redemptions

<div className="status-badge status-stable">Stable</div>

Redeem your earnings from agent usage.

## Get Redemption Balance

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/redemptions/balance</span>
</div>

Get your available balance for redemption.

### Response

```json
{
  "available": 125.5,
  "pending": 24.0,
  "lifetime": 500.0,
  "currency": "USD"
}
```

## Create Redemption

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/v1/redemptions</span>
</div>

Request a redemption.

### Request

```json
{
  "amount": 100.0,
  "method": "crypto",
  "address": "0x1234..."
}
```

### Response

```json
{
  "id": "red_abc123",
  "amount": 100.0,
  "status": "pending",
  "estimatedCompletion": "2024-01-16T10:30:00Z"
}
```

## Get Redemption Quote

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/v1/redemptions/quote</span>
</div>

Get a quote for redemption.

### Request

```json
{
  "amount": 100.0,
  "method": "crypto"
}
```

### Response

```json
{
  "amount": 100.0,
  "fee": 2.5,
  "netAmount": 97.5,
  "rate": 1.0,
  "expiresAt": "2024-01-15T10:35:00Z"
}
```

## Get Redemption Status

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/redemptions/status</span>
</div>

Check overall redemption status.

### Response

```json
{
  "eligible": true,
  "minAmount": 10.0,
  "maxAmount": 10000.0,
  "methods": ["crypto", "bank"]
}
```
