## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/elizaos/eliza.git
cd eliza
```

2. **Install dependencies**

```bash
bun install
```

3. **Set up environment**

```bash
export OPENAI_API_KEY="your-key"
```

4. **Run any example**

```bash
bun run examples/chat/typescript/chat.ts
```
