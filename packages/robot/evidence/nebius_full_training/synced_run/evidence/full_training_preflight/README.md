# End-to-End Full Training Launch Bundle

Run `scripts/00_local_preflight.sh` before copying this repo to Nebius. Scripts run from `ELIZA_ROBOT_PACKAGE_ROOT` when it is set, otherwise from the package root that generated this bundle. On the Nebius host, run scripts 10, 20, 30, 40, then 50. The generated `preflight_report.json` must have `ok: true` before launch.
