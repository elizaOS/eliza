The CLI is intentionally small. It does two things: scaffold new agent projects and scaffold new plugin packages. Everything else is driven by your project's own scripts (`bun run dev`, `bun run build`, `bun run test`).

## Install

```bash
npx elizaos --help
```

Or globally:

```bash
bun add -g @elizaos/cli
elizaos --help
```
