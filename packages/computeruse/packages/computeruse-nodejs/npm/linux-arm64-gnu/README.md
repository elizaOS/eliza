# @elizaos/computeruse-linux-arm64-gnu

This is the Linux ARM64 (glibc) platform binary for `@elizaos/computeruse`.

This package is automatically installed as a dependency when you install `@elizaos/computeruse` on Linux ARM64 with glibc.

## Requirements

- Linux ARM64 with glibc (e.g., Raspberry Pi OS 64-bit, Ubuntu ARM64)
- AT-SPI2 enabled (default on GNOME/KDE)
- `wmctrl` and `xdotool` for X11 window management

## Usage

```bash
npm install @elizaos/computeruse
```

The native binary will be automatically used when running on Linux ARM64.
