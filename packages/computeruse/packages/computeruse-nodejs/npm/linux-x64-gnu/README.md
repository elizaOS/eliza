# @elizaos/computeruse-linux-x64-gnu

This is the Linux x64 (glibc) platform binary for `@elizaos/computeruse`.

This package is automatically installed as a dependency when you install `@elizaos/computeruse` on Linux x64 with glibc.

## Requirements

- Linux x64 with glibc (most desktop Linux distributions)
- AT-SPI2 enabled (default on GNOME/KDE)
- `wmctrl` and `xdotool` for X11 window management

## Usage

```bash
npm install @elizaos/computeruse
```

The native binary will be automatically used when running on Linux x64.
