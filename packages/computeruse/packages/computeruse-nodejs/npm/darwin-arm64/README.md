# @elizaos/computeruse-darwin-arm64

This is the macOS ARM64 (Apple Silicon) platform binary for `@elizaos/computeruse`.

This package is automatically installed as a dependency when you install `@elizaos/computeruse` on macOS ARM64.

## Requirements

- macOS on Apple Silicon (M1/M2/M3)
- Accessibility permissions granted to your terminal/app

## Usage

```bash
npm install @elizaos/computeruse
```

The native binary will be automatically used when running on macOS ARM64.
