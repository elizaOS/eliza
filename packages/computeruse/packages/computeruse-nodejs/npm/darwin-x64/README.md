# @elizaos/computeruse-darwin-x64

This is the macOS x64 (Intel) platform binary for `@elizaos/computeruse`.

This package is automatically installed as a dependency when you install `@elizaos/computeruse` on macOS Intel.

## Requirements

- macOS on Intel x64
- Accessibility permissions granted to your terminal/app

## Usage

```bash
npm install @elizaos/computeruse
```

The native binary will be automatically used when running on macOS x64.
