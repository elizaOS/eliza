## Windows examples

These examples assume local Windows automation via `computeruse` is available.

### Included

- `notepad/notepad_basic.py` — Basic Notepad text editing and save
- `calculator/calculator_basic.py` — Windows Calculator automation
- `mspaint/mspaint_basic.py` — MS Paint drawing with shapes and brushes
- `screenshot/element_screenshot.py` — Element screenshot and OCR
- `screenshot/snipping_tool_freeform.py` — Snipping Tool freeform capture
- `audio/record_sound_recorder.py` — Sound Recorder automation
- `audio/playback_vlc.py` — VLC audio playback
- `video/vlc_streaming_youtube.py` — VLC network/YouTube streaming
- `menu/notepad_menu_interactions.py` — Notepad menu navigation
