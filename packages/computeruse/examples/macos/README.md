## macOS examples

macOS UI automation is **fully supported** via the native Accessibility (AX) API.

> **Requirement**: Grant Accessibility permissions to your terminal/app (System Preferences → Privacy & Security → Accessibility).

### Included

- `calculator/calculator_basic.py`
- `notepad/textedit_basic.py`
- `screenshot/element_screenshot.py`
- `audio/record_quicktime_audio.py`
- `audio/playback_vlc.py`
- `video/vlc_streaming_youtube.py`
- `menu/textedit_menu_save_as.py`
