# Virtually Ever After — Chatbot Widget

A self-contained, zero-dependency AI chatbot widget for [Virtually Ever After](https://virtuallyeverafter.xyz).

## Embed on any page

```html
<script src="https://<your-github-pages-url>/chatbot.js"></script>
```

## Local preview

```bash
npx serve vea-chatbot -p 3000
```

## Deploy

Push to `main` — GitHub Actions automatically deploys `vea-chatbot/` to GitHub Pages.
